Steps for ruuning the DoublyLinkedList.java file:

1. Compile and Run the DoublyLinkedList.java file using the below commands:
		> javac DoublyLinkedList.java
		> java DoublyLinkedList
		

Driver Program explanation:
1 => Move to the next element of the list and print it if available otherwise program ends
2 => Add a new element after the current node and moving cursor to the new node added
3 => Removing the current element and move to the previous node
4 => Move to the previous element of the current node if available otherwise throws an Exception
5 => print the list
